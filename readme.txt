При отсутствии исполняемых файлов работа не проверяется.

Если исполняемый код имеется, но возникает ошибка запуска вследствие отсутствия библиотек, то проверяющий обращается к readme файлу, где должны быть прописаны все зависимости приложения со ссылками на загрузку библиотек. По возможности докладывайте библиотеки в папку bin самостоятельно, например Qt5Core.dll, MSVCR110.dll. Для использования вашего приложения не должна требоваться установка сред разработки типа Visual Studio или Qt. В ридми должны быть прописаны используемые библиотеки, а не среды! Если проверяющий установил все библиотеки из ридми, но их оказалось недостаточно, то сдающий вызывается на апелляцию с потерей одного балла.

Если программа запускается только из среды разработки, то снимается один балл.
