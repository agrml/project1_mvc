По возможности докладывайте библиотеки в папку bin самостоятельно, например Qt5Core.dll, MSVCR110.dll. Для использования вашего приложения не должна требоваться установка сред разработки типа Visual Studio или Qt. В ридми должны быть прописаны используемые библиотеки, а не среды! Если программа запускается только из среды разработки, то снимается один балл.
